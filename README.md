# 📝 Full Stack Todo App

A modern Full Stack Todo Application built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

---

## 🚀 Features

- 🔐 User Authentication (Login & Register)
- 👤 User Profile
- ➕ Add New Tasks
- ✏️ Edit Tasks
- 🗑️ Delete Tasks
- ✅ Mark Tasks as Complete
- ↩️ Undo Completed Tasks
- 🔍 Search Tasks
- 🎯 Filter Tasks (All / Pending / Completed)
- 📅 Due Date Support
- 📊 Dashboard Statistics
- 🌙 Dark Mode
- 📱 Fully Responsive Design
- 📄 Pagination
- ⚡ Loading Spinner
- 📭 Empty State
- 💾 MongoDB Database
